# Retrieve old \`R_SEQUOIA\` \_PARCA\_ identifier

Retrieve old \`R_SEQUOIA\` \_PARCA\_ identifier

## Usage

``` r
seq1_id(dirname = ".")
```

## Arguments

- dirname:

  \`character\` Directory where the file \_PARCA\_ is located. Defaults
  to the current working directory.

## Value

The \`character\` identifier
