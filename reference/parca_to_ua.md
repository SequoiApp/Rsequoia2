# Create \_UA\_ sf object from \_PARCA\_ sf object

Create \_UA\_ sf object from \_PARCA\_ sf object

## Usage

``` r
parca_to_ua(parca)
```

## Arguments

- parca:

  \`sf\` Object from \[Rsequoia2::seq_parca()\] containing cadastral
  parcels.

## Value

An \`sf\` object containing the analysis units.
