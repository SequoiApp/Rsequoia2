# Create *UA* sf object from *PARCA* sf object

Create *UA* sf object from *PARCA* sf object

## Usage

``` r
parca_to_ua(parca)
```

## Arguments

- parca:

  `sf` Object from
  [`seq_parca()`](https://sequoiapp.github.io/Rsequoia2/reference/seq_parca.md)
  containing cadastral parcels.

## Value

An `sf` object containing the analysis units.
