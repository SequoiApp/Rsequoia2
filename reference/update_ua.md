# Update `R_SEQUOIA` *UA* layer to `Rsequoia2`

Update `R_SEQUOIA` *UA* layer to `Rsequoia2`

## Usage

``` r
update_ua(ua, id)
```

## Arguments

- ua:

  `sf` object containing analysis units

- id:

  `character` Identifier of the project

## Value

An `sf` object containing analysis units, normalized for `Rsequoia2`
use.
