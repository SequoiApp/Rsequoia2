# Compute ombrothermic climatology summaries

Computes monthly ombrothermic summaries from DRIAS extraction

## Usage

``` r
drias_ombro(txt)
```

## Arguments

- txt:

  `character` string. Path to a DRIAS `.txt` file.

## Value

A `data.frame` containing monthly averages of temperature and
precipitation for each period.

## Examples

``` r
if (FALSE) { # \dontrun{
ombro <- drias_ombro(clim)
} # }
```
