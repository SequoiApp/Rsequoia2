The goal of Rsequoia2 is to help french forest manager by retrieving and creating spatial data

## Installation

You can install the development version of Rsequoia2 from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("mucau/Rsequoia2")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(Rsequoia2)
## basic example code
```
